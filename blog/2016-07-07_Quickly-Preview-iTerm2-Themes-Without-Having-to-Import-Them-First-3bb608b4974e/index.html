Quickly Preview iTerm2 Themes Without Having to Import Them First

I was perusing https://github.com/mbadolato/iTerm2-Color-Schemes#previewing-color-schemes when all the way at the bottom, I noticed a simple tool that lets you preview themes without having to import them first!

From within the iTerm2-Color-Schemes directory, type one of the following:

  1. Apply AdventureTime scheme to the current session
tools/preview.rb schemes/AdventureTime.itermcolors
  1. Apply the schemes in turn.
  2. Press any key to advance; hit CTRL-C or ESC to stop

Preview all the themes at the same time:

tools/preview.rb schemes/*
